# 🚀 UGC Automator v2.3 — Producción Premium

El sistema más avanzado para replicar y mejorar anuncios virales de TikTok Shop utilizando IA generativa de vanguardia.

## 🎞️ Pipeline Maestro de 9 Fases
Nuestro flujo de trabajo está diseñado para maximizar la fidelidad visual y comercial:

1.  **Extracción Sementica**: `yt-dlp` descarga el vídeo; MiniMax extrae el frame ideal y describe la toma.
2.  **Face Swap (Identidad)**: `Nano Banana 2 Edit` clona la cara del influencer original con tu modelo IA.
3.  **Análisis Visual (IA Insight)**: `Wavespeed Captioner` analiza el producto físico para evitar "alucinaciones" (envase/forma).
4.  **Generación de Hook**: El **Director IA v2.3** genera un gancho comercial agresivo en Español.
5.  **Registro de Elementos**: Se crean modelos visuales (Visual Models) en Kling para el personaje y producto.
6.  **Dirección Escénica**: El Director traduce la visión a un prompt cinematográfico de +70 palabras.
7.  **Generación Kling 3.0 Pro**: Animación I2V de alta fidelidad con control de `motion` (1-10) y `cfg_scale`.
8.  **Ensamblaje de Audio**: Mix de audio trending con voces clonadas (Fase en desarrollo).
9.  **Overlay Final**: `FFmpeg` genera el vídeo listo para publicar con textos dinámicos.

## 🛠️ Stack Tecnológico
- **Frontend**: Next.js 15, Tailwind v4, Shadcn/UI (Premium Dark Mode).
- **Backend**: FastAPI (Python 3.12).
- **IA Generativa**: Wavespeed (Kling 3.0 Pro, Nano Banana 2).
- **Cerebro (Orquestador)**: MiniMax M2.7 (compatible con OpenAI).

## 🚀 Instalación y Uso
1.  Clona el repo.
2.  Configura `backend/.env` con tus claves de Wavespeed y MiniMax.
3.  Lanza el backend: `run_backend.bat`
4.  Lanza el frontend: `run_frontend.bat`
5.  Accede a `localhost:3000`.
